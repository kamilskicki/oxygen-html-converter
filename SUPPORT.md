# Support

Oxygen HTML Converter is a public beta focused on supported static marketing
and landing-page HTML. Before reporting a problem, review
[Supported Scope](docs/SUPPORTED_SCOPE.md) and the
[Upgrade Guide](docs/UPGRADE.md).

## Bugs

Use the [bug report form](https://github.com/kamilskicki/oxygen-html-converter/issues/new?template=bug_report.yml)
and include:

- the plugin version or commit
- WordPress, PHP, and Oxygen Builder versions
- the smallest reproducible HTML sample
- exact import, save, and reopen steps
- relevant converted JSON, console output, screenshots, or failure artifacts

Remove credentials, cookies, customer data, private URLs, and database content
before attaching evidence.

## Feature requests

Use the [feature request form](https://github.com/kamilskicki/oxygen-html-converter/issues/new?template=feature_request.yml).
Describe the user outcome and whether the request belongs in Core, an extension,
or a future integration.

## Security

Do not report vulnerabilities in public issues. Follow
[SECURITY.md](SECURITY.md) and use GitHub Private Vulnerability Reporting when
available.

## Oxygen Builder support

This repository supports Oxygen HTML Converter itself. Questions about Oxygen
Builder licensing, account access, or unrelated Builder behavior should be
directed to the official Oxygen support channels.
